1. Essential Features (MVP - Minimum Viable Product)
    These are the core functionalities required to use the Pomodoro technique.
    - Work Timer: A countdown timer set to 25 minutes by default.
    - Short Break Timer: A countdown timer set to 5 minutes by default.
    - Long Break Timer: A countdown timer set to 15-30 minutes, triggered after every four work sessions.
    - Start/Pause/Reset Controls: Basic ability to control the flow of the session.
    - Automatic Transition: The app should automatically switch from work to break and vice versa.
    - Sound Alert: A subtle notification sound when a session ends.

2. Basic Customization
    These features allow the user to adapt the app to their personal workflow.
    - Adjustable Durations: Ability to change the length of work, short break, and long break sessions.
    - Ability to set the number of Pomodero sessions a day
    - Visual Themes: Light and Dark mode options.
    - Session Counter: A simple visual indicator (e.g., four dots) showing how many work sessions have been completed in the current cycle.

3. Productivity & Statistics
Features that help the user understand their work habits and feel motivated.
    - Daily Log: A record of how many Pomodoros were completed today.
    - Total Focus Time: Cumulative time spent working.
    - Visual Progress Bar: A ring or bar that fills up as the timer counts down.

4. Advanced Features (Optional)
These features improve the user experience but are not strictly necessary for the technique to work.

- Desktop Notifications: Native OS notifications for when the timer ends, even if the app is minimized.
